# Data Gap Report 2026-07-02

## Summary

현재 Supabase에는 RAG chunk 380개와 `plant_catalog` 65개가 적재되어 있다. 검색/자동완성용 식물명은 넓어졌지만, 실제 관리 가이드 RAG 문서는 아직 병해충/AI Hub 라벨 요약 쪽에 치우쳐 있다.

## 확인된 부족 영역

| 식물/작물 | 현재 RAG 근거 | 문제 |
|---|---:|---|
| 몬스테라 | AI Hub 원예식물 라벨 요약 3개 | 물주기, 빛, 온도, 분갈이 같은 관리 가이드 본문 부족 |
| 스투키 | AI Hub 원예식물 라벨 요약 3개 | 건조/과습 센서 요약은 있으나 초보자 관리 문서 부족 |
| 스파티필럼 | AI Hub 원예식물 라벨 요약 3개 | 꽃/잎 처짐/광량 관리 가이드 부족 |
| 딸기 | 병해충 문서에 간접 언급 | 재배 관리, 런너, 꽃눈, 과실 관리 문서 부족 |
| 고구마 | 농사로 작목정보 간접 언급 1개 | 고구마 전용 재배 가이드 부족 |
| 장미 | 병해충 문서 간접 언급 | 가지치기, 햇빛, 개화, 화분 관리 문서 부족 |
| 벚꽃/개나리/해바라기 | 없음 | 관상식물/화훼 관리 데이터 필요 |

## 원인

1. `plant_catalog`는 이름 검색용이라 관리법을 제공하지 않는다.
2. 농사로 실내식물 상세 수집은 공개 페이지의 JS 구조 때문에 일부 상세 본문만 확보된다.
3. AI Hub 원예식물 데이터는 센서/라벨 요약에는 좋지만 사용자가 원하는 "어떻게 키워야 하는지" 가이드는 부족하다.
4. NCPMS 데이터는 병해충 참고용이라 일반 관리 질문에는 근거가 약하다.

## 우선 추가 수집 대상

### 1순위: 농사로 실내정원용 식물 상세

- 대상: 몬스테라, 스투키, 스파티필럼, 금전수, 테이블야자, 홍콩야자, 호접란, 보스턴고사리, 관음죽, 고무나무류
- 필요한 정보: 물주기, 광도, 겨울 최저온도, 습도, 생육형태, 관리 난이도
- API key: 공개 웹 수집은 불필요. OpenAPI가 있으면 `NONGSARO_API_KEY` 필요.

### 2순위: 농사로 작목정보 확장

- 대상: 딸기, 감자, 고구마, 토마토, 고추, 상추, 오이, 배추, 양배추, 파프리카, 가지, 호박
- 필요한 정보: 파종/정식, 물관리, 햇빛/온도, 생육단계, 수확, 흔한 생리장해
- API key: 공개 웹 수집은 불필요. cropEbook OpenAPI 사용 시 `NONGSARO_API_KEY` 필요.

### 3순위: 관상식물/화훼 관리 문서

- 대상: 장미, 벚나무/벚꽃, 개나리, 해바라기, 국화, 튤립, 백합, 수국, 제라늄, 베고니아
- 후보 출처: 국가생물종지식정보시스템 식물도감, 국립수목원 국가표준식물목록, 농사로 화훼/작목 자료
- 필요한 정보: 재배환경, 개화, 전정/가지치기, 월동, 화분 관리
- API key: 공공데이터포털 또는 기관 OpenAPI 사용 시 별도 인증키 필요.

## 다음 작업 제안

1. 농사로 실내식물 상세 페이지 추출기를 JS 링크 기반이 아니라 `cntntsNo` 전체 후보 탐색 방식으로 보강한다.
2. 농사로 cropEbook OpenAPI operation 명세와 `NONGSARO_API_KEY`를 확보해 작물별 전자책 목차/본문을 수집한다.
3. `plant_catalog`에 있는 65종 중 RAG 관리 문서가 없는 항목을 `coverage_status=missing_care_doc`로 표시하는 검증 스크립트를 추가한다.
4. 관상식물은 `ornamental_care` 카테고리로 별도 수집해 병해충 문서와 섞이지 않게 한다.
