# Interim Data

수집 후 정규화 전/후의 중간 JSONL을 저장하는 폴더입니다.

## 규칙

- 실제 중간 산출물은 Git에 커밋하지 않습니다.
- 재현 가능한 스크립트와 schema만 Git에 남깁니다.

## 예시

```text
data/interim/web_documents.jsonl
data/interim/ncpms_pest_reference.jsonl
data/interim/psis_pesticide_safety.jsonl
data/interim/rag_documents.normalized.jsonl
```
